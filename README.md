### Cara menjalankan project

#### 1. Clone atau download project
#### 2. Buat database crud pada localhost
#### 3. Jalankan file sql yang ada di folder project pada database crud
#### 4. Buka folder project dan jalankan di text editor
#### 5. Buka terminal dan jalankan npm run dev
